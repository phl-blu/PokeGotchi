# Commands to Create All Pokemon Egg Folders

Run these commands to create the correct folder structure:

```bash
# Create egg folders for all 9 Pokemon lines
mkdir -p src/main/resources/pokemon/sprites/charmander/egg/{stage1,stage2,stage3,stage4}
mkdir -p src/main/resources/pokemon/sprites/cyndaquil/egg/{stage1,stage2,stage3,stage4}
mkdir -p src/main/resources/pokemon/sprites/mudkip/egg/{stage1,stage2,stage3,stage4}
mkdir -p src/main/resources/pokemon/sprites/piplup/egg/{stage1,stage2,stage3,stage4}
mkdir -p src/main/resources/pokemon/sprites/snivy/egg/{stage1,stage2,stage3,stage4}
mkdir -p src/main/resources/pokemon/sprites/froakie/egg/{stage1,stage2,stage3,stage4}
mkdir -p src/main/resources/pokemon/sprites/rowlet/egg/{stage1,stage2,stage3,stage4}
mkdir -p src/main/resources/pokemon/sprites/grookey/egg/{stage1,stage2,stage3,stage4}
mkdir -p src/main/resources/pokemon/sprites/fuecoco/egg/{stage1,stage2,stage3,stage4}
```

Each Pokemon will have its own unique egg design:
- **Charmander**: Orange/red fire-themed egg
- **Cyndaquil**: Blue/yellow fire-themed egg  
- **Mudkip**: Blue water-themed egg
- **Piplup**: Blue/white water-themed egg
- **Snivy**: Green grass-themed egg
- **Froakie**: Blue/white water-themed egg
- **Rowlet**: Brown/green grass/flying-themed egg
- **Grookey**: Green/brown grass-themed egg
- **Fuecoco**: Red/orange fire-themed egg
